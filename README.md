# 🕊️ Агент по паллиативной помощи

**Суть проекта:**
Это интеллектуальный помощник, созданный для информационной поддержки людей, столкнувшихся с необходимостью паллиативного ухода. 
В моменты сильного стресса пациентам и их близким тяжело искать разрозненную медицинскую и юридическую информацию. 
Агент решает эту проблему, объединяя все нужные данные в одном месте с удобным поиском.

**Основные функции:**
* 🏥 **Навигация по учреждениям:** Быстрый поиск актуальных адресов, контактов и профилей хосписов.
* 📄 **Сбор документов:** Четкие списки необходимых справок и бумаг без бюрократической воды.
* ⚖️ **Алгоритмы действий:** Пошаговые инструкции о том, куда обращаться, как оформить статус и получить помощь.

Главная цель продукта — снизить информационную нагрузку на семьи и предоставить быстрый, понятный и точный доступ к жизненно важной информации.
